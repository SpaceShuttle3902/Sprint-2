# Peg Solitaire Game – Sprint 2

This project is a Python implementation of the classic *Peg Solitaire (BrainVita)* game developed for **CS 449 – Software Engineering**.

It features a graphical user interface built with *Tkinter* and follows a modular design by separating the game logic from the user interface.

The application allows users to select board size and type (English, Diamond, Hexagon), start a new game, make valid moves, and automatically detect when no valid moves remain.

Automated unit tests using *unittest* verify board initialization, valid moves, and game-over detection.

*Note:* The game-over logic works correctly and is tested, but there is a minor GUI issue where the pop-up message may not always display.
